This engine includes source code and algorythms from several One Lone Coder master classes and products, specifically:

	- Pixel Game Engine: PixFu is inspired on OLC/PGE loop, refactored to use dynamic extensions, several opengl layers and modular input devices. Geometric Graphics functionality, and Keyboard and Mouse objects are refactored versions of the ones in OLC/PGE.

	- Circles, Balls & Collisions I && II: An awesome masterclass on how to implement ball physics and collision detection.

	- Splines I & II: Another awesome masterclass on magic with splines.

	- Apple Cocoa Initialization: The apple platform layer from OneLoneCoder Mac Port.

	Please follow jdavidx YouTube Channel so we can keep learning new tricks !!!
